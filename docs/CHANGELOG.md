## [0.1.0] — 2026-03-31

### Added
- Initial project scaffold
- Natural language time zone query parser (regex-based)
- Time conversion across preset zones: Adelaide, Manila, Palau, Guam
- Single-window SwiftUI interface with text input and results list
- Day offset indicator (+1d / -1d) for cross-date conversions
- Source and destination zone highlighting
- Unit tests for parser and conversion service
- SwiftLint + SwiftFormat tooling
- Makefile build automation
